ВЫПОЛНЕННЫЕ ТРЕБОВАНИЯ:
------------------------------------------
1. ✅ Аутентификация и авторизация
   - JWT токены
   - Три уровня доступа (админ, модератор, пользователь)

2. ✅ Модели данных
   - Пользователь (Custom User)
   - Курс (Course)
   - Урок (Lesson)
   - Платеж (Payment)
   - Подписка (Subscription)

3. ✅ API Endpoints
   - /api/v1/materials/courses/ - CRUD курсов
   - /api/v1/materials/lessons/ - CRUD уроков
   - /api/v1/materials/subscription/ - подписки
   - /api/v1/users/payments/ - платежи
   - /api/v1/users/profile/ - профиль

4. ✅ Бизнес-логика
   - Валидация YouTube ссылок
   - Фильтрация по пользователю
   - Пагинация результатов
   - Права доступа к контенту

5. ✅ Документация
   - README.md
   - INSTRUCTIONS.md
   - Swagger документация

ИНСТРУКЦИЯ ПО ЗАПУСКУ:
------------------------------------------
1. Установка: pip install -r requirements.txt
2. Миграции: python manage.py migrate
3. Тестовые данные: python manage.py shell
   (выполнить код из create_test_data.py)
4. Запуск: python manage.py runserver
5. Тестирование: http://localhost:8000/api/v1/
